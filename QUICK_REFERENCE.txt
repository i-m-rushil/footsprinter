┌──────────────────────────────────────────────────────────────────────────────┐
│ BASIC USAGE                                                                  │
└──────────────────────────────────────────────────────────────────────────────┘

  Quick Scan:
  $ sudo ./footsprinter.sh --url example.com

  Full Scan:
  $ sudo ./footsprinter.sh --url example.com --fullscan

  Stealth Scan:
  $ sudo ./footsprinter.sh --url example.com --interval 3 --changeheaders

┌──────────────────────────────────────────────────────────────────────────────┐
│ PARAMETERS                                                                   │
└──────────────────────────────────────────────────────────────────────────────┘

  --url <domain>        Target domain (REQUIRED)
  --fullscan            Enable comprehensive deep scan
  --interval <seconds>  Delay between requests (default: 1)
  --changeheaders       Randomize User-Agent headers
  -h, --help            Show help message

┌──────────────────────────────────────────────────────────────────────────────┐
│ WHAT IT SCANS                                                                │
└──────────────────────────────────────────────────────────────────────────────┘

  ✓ Company Intelligence (WHOIS, DNS, IP geolocation)
  ✓ Subdomains (Subfinder, Assetfinder, Amass, crt.sh)
  ✓ Live Host Detection (httpx)
  ✓ Port Scanning (nmap - TCP/UDP)
  ✓ Technology Stack (WhatWeb, httpx tech-detect)
  ✓ Vulnerability Assessment (Nuclei 3000+ templates)
  ✓ Web Server Vulnerabilities (Nikto in fullscan)
  ✓ URL Harvesting (Wayback, GAU, Katana)
  ✓ Misconfiguration Checks
  ✓ Risk Assessment & Scoring

│ TROUBLESHOOTING                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

  Permission Denied:        chmod +x footsprinter.sh
  Tool Not Found:           Run with sudo, auto-installs dependencies
  Slow Scan:                Remove --fullscan or increase --interval
  No Vulnerabilities:       Good! Or try --fullscan for deeper check
  Python Error:             sudo apt install python3 python3-pip
